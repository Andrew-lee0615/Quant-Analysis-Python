# 📈 Quant-Analysis-Python

A collection of financial analysis tools and risk modeling scripts.

## 🚀 Project 1: Maximum Drawdown (MDD) Tracker
### **Analysis Objective**
In light of the potential 'Liquidity Void' discussed in recent market outlooks, this tool calculates the historical peak-to-trough decline (MDD) to quantify tail risk.

### **Tech Stack**
* **Language:** Python
* **Libraries:** Pandas (Data Handling), NumPy (Math Operations)
* **Environment:** Google Colab

### **Key Insights**
* Developed a vectorized operation using `.cummax()` to efficiently track historical peaks.
* Calculated MDD to identify the "worst-case scenario" for risk management frameworks.
